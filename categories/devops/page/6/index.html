Drone 發佈 0.8.0-rc.1 版本

 Posted on July 21, 2017  |  1 minutes  |  72 words  |  appleboy

Drone 作者在昨天晚上發佈了 0.8.0-rc.1，此版本有兩個重大變更，第一是 Server 跟 Angent 之間溝通方式轉成 GRPC，另一個變更則是將原本單一執行擋 drone 拆成兩個，也就是之後會變成 drone-server 及 drone-agent，拆成兩個好處是，通常 Server 端只會有一台，但是隨著專案越來越多，團隊越來越龐大，Agent 肯定不只有一台機器，所以把 Agent 拆出來可以讓維運人員架設新機器時更方便。

Ruby 開發環境，在 Ubuntu 或 Debian 底下可以透過 apt 的方式安裝:

$ sudo apt-get install ruby1.9.1-full
如果你想裝 ruby 1.8 可以透過底下:
$ sudo apt-get install ruby-full

  * [如何正確實作出固定寬度且置中的版型][8]
  * [CSS 排版觀念：Position][9]
  * [樂多Blog排版入門][10] 
  * [CSS 語法教學][11]
  * [CSS 排版觀念：Float][12]
  * [CSS 排版觀念：Box Model][13]  可以在搭配 

[jQuery][14] 來控制 CSS 屬性喔，CSS 可以利用 [FireFox][15] 的 [Firebug][16] 來除錯喔，這樣其實就很簡單了吧。

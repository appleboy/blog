  * 如果你有 github 帳號，請把帳號給我，我會加入共同作者，這樣就可以直接 commit merge checkout pull push 進去 repository
  * 把 patch 檔案寄給我，diff -u 原檔名 翻譯後檔名  > 檔名.patch
  * 或者是將片段翻譯，寄給我，並告知識翻譯哪個部份這樣就可以了 我的聯絡方式：

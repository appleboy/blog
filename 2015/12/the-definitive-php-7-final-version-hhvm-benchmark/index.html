PHP 7 vs HHVM Benchmark 比較

 Posted on December 9, 2015  |  1 minutes  |  53 words  |  appleboy

2015-12-03 PHP 終於釋出 7.0 版本，kinsta 工程師馬上寫出一篇 PHP7 vs HHVM 效能比較文章，直接先講結論，就是 HHVM 效能還是大於 PHP 7，所以 kinsta 最終還是採用 HHVM 來 Host WordPress 網站，文章內容都是以 CMS 平台做比較，而不是以各大 Framework 來比較，但是看結果來說，HHVM > PHP7 >> PHP 5，所以建議可以升級到 PHP 7 或者是直接上 HHVM 也可以了。底下是 WordPress benchmark 比較圖。

WordPress 效能看起來非常相近，如果是測試 Drupal 8.0.1 效能差距會比較大 (Drupal 每分鐘線上 15 人)

要升級到 7.0 的開發者，請先看完 PHP Migration Guide。

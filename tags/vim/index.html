mkdri ~/.vim

Ctrl+w 2次則會切換到令一個視窗

fllow mode

3.去除 ^H ^M
man gcc | col -b > gcc.txt

vi 後 :1,$s/^v^H//g  (^v = ctrl + v, ^H = ctrl+h)

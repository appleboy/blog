解決在 Ubuntu 系統下 Chrome 瀏覽器亂碼

 Posted on December 10, 2014  |  1 minutes  |  25 words  |  appleboy

在 Ubuntu 用 Chrome 瀏覽器一陣子後，突然發現全部的中文介面都變成亂碼，就像上面這張截圖一樣，除了網頁可以正常顯示外，其他像是 Tab 或書籤都變成亂碼，在百度找到這篇解答，解決方式很容易，但是就是不知道原因為什麼這樣改就可以，底下紀錄如何解決此問題。

打開 /etc/fonts/conf.d/49-sansserif.conf 可以看到底下內容




        
                
                        sans-serif
                
                
                        serif
                
                
                        monospace
                
                
                        sans-serif

找到倒數第四行的 sans-serif 換成 ubuntu 字串，關閉瀏覽器重新開啟即可。
